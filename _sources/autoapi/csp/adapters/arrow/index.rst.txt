.. py:data:: CRecordBatchPullInputAdapter

   Stream record batches using the PyCapsule C Data interface from an iterator/generator into csp

   :param ts_col_name: Name of the timestamp column containing timestamps in ascending order
   :param source: Iterator/generator of pycapsule objects obtained by calling __arrow_c_array__() on the python record batches
   :param schema: The schema of the record batches
   :param expect_small_batches: Optional flag to optimize performance for scenarios where there are few rows (<10) per timestamp

   :returns: List of pycapsule objects each corresponding to a record batch in the PyCapsule representation (similar to calling __arrow_c_array__() on a record batch)

   NOTE: The ascending order of the timestamp column must be enforced by the caller

.. py:function:: RecordBatchPullInputAdapter(ts_col_name: str, source: Iterable[pyarrow.RecordBatch], schema: Optional[pyarrow.Schema] = None, expect_small_batches: bool = False) -> csp.ts[[pyarrow.RecordBatch]]

   Stream record batches from an iterator/generator into csp

   :param ts_col_name: Name of the timestamp column containing timestamps in ascending order
   :param source: Iterator/generator of record batches
   :param schema: Schema of the record batches. If None, extracted from first batch at runtime.
   :param expect_small_batches: Optional flag to optimize performance for scenarios where there are few rows (<10) per timestamp

   NOTE: The ascending order of the timestamp column must be enforced by the caller


.. py:function:: write_record_batches(where: str, batches: csp.ts[List[pyarrow.RecordBatch]], kwargs: dict, merge_record_batches: bool = False, max_batch_size: int = 0)

   Dump all the record batches to a parquet file

   :param where: destination to write the data to
   :param batches: The timeseries of list of record batches
   :param kwargs: additional args to pass to the ParquetWriter
   :param merge_record_batches: A flag to combine all the record batches in a single tick into a single record batch
   :param max_batch_size: the max size of each batch to be written, combine record batches across ticks, does not split record batches. So if a record batch is larger than max_batch_size, it is written as is.

   Convert ts[List[pa.RecordBatch]] into ts[List[T]] where T is a csp.Struct type.

   :param data: Timeseries of lists of Arrow RecordBatches
   :param cls: Target csp.Struct type
   :param schema: Arrow schema of the record batches (required).
   :param field_map: Optional mapping of struct field name -> arrow column name.
                     If None, defaults to identity naming for all fields.
   :param numpy_dimensions_column_map: Optional mapping of arrow column name -> dimensions column name
                                       for NumpyNDArray fields. If not provided for an NDArray field, defaults to
                                       ``<column_name>_csp_dimensions``.

   :returns: Timeseries of lists of struct instances

   Convert ts[List[T]] into ts[List[pa.RecordBatch]] where T is a csp.Struct type.

   :param data: Timeseries of lists of struct instances
   :param cls: Source csp.Struct type
   :param field_map: Optional mapping of struct field name -> arrow column name.
                     If None, defaults to identity naming for all fields.
   :param numpy_dimensions_column_map: Optional mapping of arrow column name -> dimensions column name
                                       for NumpyNDArray fields. If not provided for an NDArray field, defaults to
                                       ``<column_name>_csp_dimensions``.
   :param max_batch_size: Maximum number of rows per output RecordBatch.
                          Defaults to 65536. Set to 0 to disable chunking.

   :returns: Timeseries of lists of RecordBatch
