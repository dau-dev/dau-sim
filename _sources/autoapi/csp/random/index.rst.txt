.. py:function:: poisson_timer(rate: csp.ts[float], seed: object, value: ~T = True) -> csp.ts[T]

   Generate events according to a Poisson process with time-varying rate.
   For a fixed-interval timer see csp.timer.

   :param rate: The rate of the Poisson process (per second), must be non-negative
   :param seed: The seed for the numpy random Generator. Can be anything accepted by np.random.default_rng
   :param value: The value to tick when there are events (similar to csp.timer)


.. py:function:: brownian_motion(trigger: csp.ts[object], drift: csp.ts[csp.typing.Numpy1DArray[float]], covariance: csp.ts[csp.typing.NumpyNDArray[float]], seed: object, return_increments: bool = False) -> csp.ts[csp.typing.Numpy1DArray[float]]

   Generate multi-dimensional Brownian motion (or increments) at the trigger times, with time-varying drift and covariance.
   The Brownian motion starts once drift and covariance have at least 1 tick each, and will start from zero.
   To use a different start value, use csp.const(initial_value) + brownian_motion(...)

   :param trigger: When to return the value of the process
   :param drift: Drift parameter (per second), i.e. array of length n
   :param covariance: Covariance matrix (per second), i.e. array of size nxn
   :param seed: The seed for the numpy random Generator. Can be anything accepted by np.random.default_rng
   :param return_increments: Whether to return increments of the brownian motion at trigger times instead of the process itself


.. py:function:: brownian_motion_1d(trigger: csp.ts[object], drift: csp.ts[float], variance: csp.ts[float], seed: object, return_increments: bool = False) -> csp.ts[float]

   Generate one-dimensional Brownian motion at the trigger times, with time-varying drift and variance.
   The Brownian motion starts once drift and covariance have at least 1 tick each, and will start from zero.
   To use a different start value, use csp.const(initial_value) + brownian_motion_1d(...)

   :param trigger: When to return the value of the process
   :param drift: Drift parameter (per second)
   :param variance: Variance parameter (per second)
   :param seed: The seed for the numpy random Generator. Can be anything accepted by np.random.default_rng
   :param return_increments: Whether to return increments of the brownian motion at trigger times instead of the process itself
