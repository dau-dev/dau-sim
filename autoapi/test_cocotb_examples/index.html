test_cocotb_examples¶

End-to-end tests based on cocotb’s official examples.

Each test translates a cocotb example HDL design into dau-sim IR, sets up the cocotb scheduler with a patched simulator module, and runs the same async test patterns used in the upstream cocotb examples — verifying that dau-sim drives the cocotb testbench correctly without any intermediary compilation step.

Covered examples: - simple_dff: D flip-flop with clock-driven sequential test - adder: Purely combinational adder with reference-model checks - simple_counter: Counter with rst/ena/set/din, multi-coroutine tests

test_cocotb_examples._make_simple_counter()¶

8-bit counter with rst, ena, set, din.

Equivalent to cocotb/examples/doc_examples/quickstart/simple_counter.sv:

always_ff @(posedge clk) begin
    if (rst)          counter <= 0;
    else if (set)     counter <= din;
    else if (ena)     counter <= counter + 1;
    else              counter <= counter;
end
