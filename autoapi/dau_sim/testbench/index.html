force(signal: str, value: int) → None[source]¶

Force a signal to a constant value, overriding simulation output.

The forced value is applied after each tick() and persists until release() is called.

release(signal: str) → None[source]¶

Release a previously forced signal, restoring simulation control.

is_forced(signal: str) → bool[source]¶

Return True if the signal is currently forced.

tick(n: int = 1) → None[source]¶

Advance simulation by n clock cycles.

Runs the compiled module for n cycles using the current signal values as initial state. After execution, all signal values are updated to reflect the final simulated state.

Assert that signal has been stable (unchanged).

If cycles is given, checks that the last cycles recorded values are identical. Otherwise checks all recorded history.

dau_sim.testbench.run_parallel_testbenches(compiled_module, *fns, clock_period: datetime.timedelta = timedelta(microseconds=1), clocks: dict[str, datetime.timedelta] | None = None, max_cycles: int = 10000) → TestbenchResult[source]¶

Run multiple testbench functions in parallel against a compiled module.

Each function receives a _ParallelView that supports the same API as TestbenchContext. All threads synchronize at each tick() call so the simulation advances in lock-step.

Returns a TestbenchResult capturing the final state.

Raises the first exception from any thread if a testbench fails.
