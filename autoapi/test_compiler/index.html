class test_compiler.TestMemoryExecution¶

Tests for memory read/write execution in compiled simulation.

_make_mem_module(*, init=(), depth=4, comb_read=True)¶

Build a module with one 8-bit memory, 1 write port, 1 read port.

Signals: wr_addr, wr_data, wr_en (write), rd_addr, rd_data (read). Write domain = ‘sync’, read domain = None (comb) or ‘sync’.

test_memory_init_values()¶

test_memory_read_init_data()¶

Combinational read should return init data from cycle 1.

test_memory_sync_read()¶

Synchronous read port should latch data on clock edge.

test_memory_granular_write()¶
