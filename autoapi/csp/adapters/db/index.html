class csp.adapters.db.DBReader(connection, time_accessor, table_name=None, schema_name=None, query=None, symbol_column=None, constraint=None, log_query=False, use_raw_user_query=False)[source]¶
Parameters:

  • connection – sqlalchemy engine or (already connected) connection object.

  • time_accessor – TimeAccessor object

  • table_name – name of table in database as a string (table only, do not include database and schema)

  • schema_name – name of the schema to use (for databases like SQL Server that use schema as part of table location, may not be needed if schema name is “dbo”)

  • query – either string query or sqlalchemy query object. Ex: “select * from users”

  • symbol_column – name of symbol column in table as a string

  • constraint – additional sqlalchemy constraints for query. Ex: constraint = db.text( ‘PRICE>:price’ ).bindparams( price = 100.0 )

  • log_query – set to True to see what query was generated to access the data

  • use_raw_user_query – Don’t do any alteration to user query, assume it contains all the needed columns and sorting
